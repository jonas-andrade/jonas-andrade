## Sobre

Construo simuladores corporativos imersivos — ambientes 3D isométricos onde profissionais vivem um dia de trabalho real antes de entrar no mercado. Arquitetura de software, produto e infraestrutura são minha responsabilidade na Infinitra.

Fora do produto: apaixonado por arquiteturas web robustas, Linux, networking e sistemas embarcados. Comecei analisando falhas de autenticação em redes móveis usando Nokia Asha 303 e Opera Mini — curiosidade técnica é o que me move.

## O que estou construindo

### Infinitra Systems · CTO & Co-founder
> Plataforma de simulação corporativa imersiva para treinamento profissional.  
> Ambientes 3D isométricos com NPCs autônomos, pressão de tempo real e avaliação normativa automática.

**Stack do produto:** React · Three.js · React Three Fiber · Zustand · Node.js · GLTF · Web Audio API

O simulador replica um dia completo de trabalho — com chamados reais, SLAs, incidentes inesperados e consequências jurídicas mensuradas por normas como ITIL 4, LGPD e ISO 27001. O profissional erra no simulador antes de errar na empresa.
